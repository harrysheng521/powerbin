搜索框

使用$.fn.searchbox.defaults重载默认值。

依赖关系

  • 菜单按钮

使用方法

创建搜索框

1. 使用标签创建。对input标签引用'kjaxui-searchbox'类。

	<script type="text/javascript">
		function qq(value,name){
			alert(value+":"+name)
		}
	</script>
	<input id="ss" class="kjaxui-searchbox"
			searcher="qq"
			prompt="Please Input Value" menu="#mm" style="width:300px"></input>
	<div id="mm" style="width:120px">
		<div name="all" iconCls="icon-ok">All News</div>
		<div name="sports" >Sports News</div>
	</div>

2. 使用脚本创建。

	<input id="ss" ></input>
	<div id="mm" style="width:120px">
		<div name="all" iconCls="icon-ok">All News</div>
		<div name="sports" >Sports News</div>
	</div>
	$('#ss').searchbox({
		width:200,
		searcher:function(value,name){
			alert(value + "," + name)
		},
		menu:'#mm',
		prompt:'Please Input Value'
	});

2. 当点击搜索按钮需要弹框时，做如下配置即可。

<input id="search" class="wsui-searchbox" data-options="dialogOptions:{height:440,resizable:true,href:'xxx',param:{dialogId:'dd',queryParams:'#formid'}}"/> 当需要弹框时请在data-options里面配置dialogOptions属性，dialogOptions里面可以配置dialog的所有属性。 param属性为需要传递的参数，dialogId为弹出框的id，为方便在弹出页面中操作dialog，请配置dialogId, param里面的queryParams属性为弹出页面所需要的参数，可以配置函数名，例如queryParams:'getData',function getData(){} getData()函数返回一个json对象，例如{"name":"searchbox","age":"21"}。 queryParams还支持配置表单id，例如：queryParams:'#formid',form表单下的数据会被当成参数传递到目标页面。 queryParams支持配置对象，例如：queryParams:{name:'searchbox',age:'21'}。 在目标页面通过${param.属性明}来获取参数，例如${param.dialogId}取到dialogId属性的值

属性

名称 类型 描述 默认值
width（宽度） number（数字） 设置组建的宽度。 auto
propmt（提醒） string（字符串） 显示在搜索框的提醒信息。 ''
value（值） string（字符串） 搜索框的默认值。 ''
menu（菜单） selector（选择器） 搜索类型下拉菜单。 

<input class="kjaxui-searchbox" style="width:300px" data-options="menu:'#mm'" />
<div id="mm" style="width:150px">
  <div data-options="name:'item1'">Search Item1</div>
  <div data-options="name:'item2',selected:true">Search Item2</div>
  <div data-options="name:'item3'">Search Item3</div>
</div>
null
searcher（搜索器） function(value,name) 当用户点击搜索按钮或者按下ENTER键的时候搜索函数将被调用。 null

方法

名称 参数 描述
options none 返回搜索框属性对象。
menu none 返回搜索类型菜单对象。 

var m = $('#ss').searchbox('menu');  // get the menu object
var item = m.menu('findItem', 'Sports News');  // find the menu item
// change the menu item icon
m.menu('setIcon', {
	target: item.target,
	iconCls: 'icon-save'
});
// select the searching type name
$('#ss').searchbox('selectName', 'sports');
textbox none 返回文本框对象。
getValue none 返回当前搜索关键字。
setValue value 设置新的搜索关键字。
getName none 返回当前搜索类型。
selectName name 选中当前搜索类型。 

$('#ss').searchbox('selectName', 'sports');
destroy none 清除搜索框组件
resize width 重置搜索框组建的宽度。
