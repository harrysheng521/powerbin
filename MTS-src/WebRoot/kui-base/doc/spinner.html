微调器

使用$.fn.spinner.defaults重载默认值

依赖关系

  • validatebox
  • 继承spinner

使用方法

创建微调器

使用html标签创建微调器，只须要对<input>标签引用'easyui-spinner'类。

例子：

	<input id="ss" value="2"">


	
	$('#ss').spinner({    
		required:true,    
		increment:10
	});

 

属性

名称 类型 描述 默认值
width（宽度） number（数字） 可伸缩面板所在容器的宽度。 auto
height（高度） number（数字） 可伸缩面板所在容器的高度。 auto
value string 微调值器的初始值
min string 微调值的最小值 null
max string 微调值的最大值 null
increment number 自增长步长 1
editable boolean 是否可编辑 true
disabled boolean 是否禁用输入框 false
separator 分隔符 分隔符，默认冒号 :
showSeconds boolean 是否显示秒钟 false
highlight 高亮显示，0表示小时，1表示分钟，2表示秒 0

事件

名称 参数 描述
onSpinUp none 点击向上调节按钮时触发
onSpinDown none 点击向下调节按钮时触发

方法

名称 参数 描述
options none 返回属性对象
destroy none 销毁微调器组件。
resize width 重置微调器 

$('#ss').spinner('resize');  // 重置为微调器的初始宽度
$('#ss').spinner('resize', 200);  // 设置微调器为新的宽度200
enable none 使微调器可用
disable none 使微调器不可用
getValue none 获取微调器的值
setValue value 设置微调器的值
clear none 清除微调器的值
spin down 回调方法
setValue value 设值
getHours none 获得小时值
getMinutes none 获得分钟值
getSeconds none 获得秒值
