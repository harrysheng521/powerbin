金额表单

继承$.fn.validatebox.defaults，使用$.fn.moneybox.defaults重载默认值。

使用方法

使用HTML标签创建金额表单，只须要对<div>标签引用'kjaxui-moneybox'类。

	<input class="kjaxui-moneybox" data-options="required:true,validType:'money[1,10]'"/>

用javaScript来创建表单验证

	<input id="vv" required="true" validType="money[1,10]">

	$('#vv').moneybox({
		required:false
	});

属性

名称 类型 描述 默认值
required（必填） boolean（布尔型） 定义表单域必须填写。 false
validType（验证类型） string（字符串） 定义表单域的验证类型，比如money等。 null
groupSeparator String 数字之间的分隔符。三位一分隔 ,
disabled boolean 设置为true时。表单不可用。 false.
maxLength String 设置的值，为表单输入的最大长度。

方法

名称 参数 描述
destroy none 移除并注销组件。
disable none 设置表单不可用
enable none 设置表单可用
clear none 清除表单数据。
