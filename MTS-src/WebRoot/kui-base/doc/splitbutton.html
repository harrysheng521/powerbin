分隔按钮

继承$.fn.linkbutton.defaults，使用$.fn.splitbutton.defaults重载默认值。

依赖关系

  • 菜单
  • 分隔按钮

使用方法

创建分隔按钮

使用HTML标签创建分隔按钮，只须要对<div>标签引用'kjaxui-splitbutton'类。

  
	<a href="javascript:void(0)" id="sb" class="kjaxui-splitbutton"  data-options="menu:'#mm'" 
		iconCls="icon-ok" onclick="javascript:alert('ok')">Ok</a>
	  <div id="mm" style="width:100px;">
	  <div iconCls="icon-ok">Ok/div>
	  <div iconCls=""icon-cancel">Cancel/div>
	</div>

在<div>标签上创建分隔按钮。

	<a href="javascript:void(0)" id="sb" iconCls="icon-ok" onclick="javascript:alert('ok')">Ok</a>
	  <div id="mm" style="width:100px;">
	  <div iconCls="icon-ok">Ok/div>
	  <div iconCls=""icon-cancel">Cancel/div>
	</div>

	$('#sb').splitbutton({
	  menu:'#mm'
	});

Properties

属性继承链接按钮, 以下是分隔按钮的独有属性。

名称 类型 描述 默认值
plain（简洁模式） boolean（布尔型） 设置为true将显示简洁效果。 true
menu（菜单） string（字符串） 用来创建一个相应菜单的选择器。 null
duration（持续时间） number（数字） 定义鼠标划过按钮时显示菜单所持续的时间，单位为毫秒。 100

方法

名称 参数 描述
options none 返回属性对象。
disable none 禁用分隔按钮。 

$('#sb').splitbutton('disable');
enable none 启用分隔按钮。 

$('#sb').splitbutton('enable');
destroy none 清除分隔按钮。
