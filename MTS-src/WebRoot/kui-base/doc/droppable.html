拖动至容器

使用$.fn.droppable.defaults重载默认值。

使用方法

创建拖动至容器

1. 使用标签创建。对<div>标签引用'kjaxui-droppable'类。

	<div id="dd" class="kjaxui-droppable" data-options="accept:'#d1,#d3'" style="width:100px;height:100px;"></div>

2. 使用脚本创建。

	<div id="dd" style="width:100px;height:100px;"></div>

	$('#dd').droppable({
		accept:'#d1,#d3'
	});

属性

名称 类型 描述 默认值
accept selector 决定哪个可拖动元素将会被接收。 null
disabled boolean 设置true将无法拖动元素 false

事件

名称 参数 描述
onDragEnter e,source 当可拖动元素被拖入目标容器的时候触发，参数source是被拖动的DOM元素。
onDragOver e,source 当可拖动元素被拖至某个元素之上的时候触发，参数source是被拖动的DOM元素。
onDragLeave e,source 当可拖动元素被拖离目标容器的时候触发，参数source是被拖动的DOM元素。
onDrop e,source 当可拖动元素被拖入目标容器并放置的时候触发，参数source是被拖动的DOM元素。

方法

名称 参数 描述
options none 返回拖动至容器的属性
enable none 设置拖动至容器可用
disable none 设置拖动至容器不可用
