表单验证

继承$.fn.validatebox.defaults，使用$.fn.textbox.defaults重载默认值。

使用方法

使用HTML标签创建表单验证，只须要对<div>标签引用'kjaxui-textbox'类。

	<input class="kjaxui-textbox" data-options="required:true,validType:'length[1,10]'"/>

用javaScript来创建表单验证

	<input id="vv" required="true" validType="email">

	$('#vv').textbox({
		required:true
	});

验证规则

验证规则使用required和validType属性来定义, 以下列出的是插件内置的验证规则。

  • email: 正则匹配电子邮件。
  • url: 正则匹配url。
  • length[0,100]: 验证长度范围。
  • remote['http://.../action.do','paramName']: 发送ajax请求来验证，验证有效时返回true。
  • mobile:正则匹配手机号码
  • tel:正则匹配普通电话，传真号码
  • numchar[0,100]:正则匹配只含有数字或者英文字母，长度是0到100个字符
  • val[0,100]:正则匹配验证只含有数字或者英文字母或者汉字，长度是0到100个字符，但是能输入.和-_
  • number[4,2]:正则匹配任意数字,4为小数点前位数,2为小数点后位数
  • int:正则匹配整数（含0）
  • zint:正则匹配正整数（不含0）
  • num[4]:正则匹配确切位数数字，4为多少位数字
  • unhan[100]:正则匹配非汉字，100为最大值。
  • postcode:正则匹配邮政编码。
  • cardno:正则匹配身份证号码。

属性

名称 类型 描述 默认值
required（必填） boolean（布尔型） 定义表单域必须填写。 false
validType（验证类型） string（字符串） 定义表单域的验证类型，比如：email, url等。 null
showMessage boolean 设置为true时。显示提示信息。 false.
disabled boolean 设置为true时。表单不可用。 false.
maxLength String 设置的值，为表单输入的最大长度。

方法

名称 参数 描述
destroy none 移除并注销组件。
validate none 验证表单域的内容是否有效。
isValid none 调用validate方法并且返回验证结果，true或者false。
disable none 设置表单不可用
enable none 设置表单可用
clear none 清除表单数据。
