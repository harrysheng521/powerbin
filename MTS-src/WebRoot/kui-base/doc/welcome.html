文档说明

每个kui组件都有属性，方法，事件。

属性

事件

绑定事件语法：$('selector').{comp组件名称}({event:function(){・・・}});

方法

调用方法语法: $('selector').{comp组件名称}('method', parameter);

Where:

  • selector jQuery对象选择器。
  • comp 组件名。
  • method 该组件存在的方法。
  • parameter 参数，可以为对象，字符串・・・

方法都定义在jQuery.fn.{comp}.methods. 每个方法都有两个参数：jq 和 param. jq参数是必须的，指jQuery对象. 第二个参数param指真正传给方法的参数. 例如，为对话框插件拓展一个名为mymove的方法，代码如下： 

$.extend($.fn.dialog.methods, {
    mymove: function(jq, newposition){
        return jq.each(function(){
            $(this).dialog('move', newposition);
        });
    }
});

现在你可以调用mymove方法去移动对话框到特定的位置：

$('#dd').dialog('mymove', {
    left: 200,
    top: 100
});

开始应用 kui

下载kui库并添加kui CSS和JavaScript文件到你的页面上. 

<link rel="stylesheet" typ="text/css" href="../../kui-base/themes/icon.css">
<script type="text/javascript" src="../../kui-base/js/core/jquery-1.8.3.min.js"></script>
<script type="text/javascript" src="../../kui-base/js/core/kui.loader.js">></script>

只要将kui必须的文件都添加到你的页面，你就可以通过标签或JS来定义一个kui组件。 例如： 定义一个可以折叠的面板，HTML代码如下： 

<div id="p" class="kui-panel" style="width:500px;height:200px;padding:10px;"
	 title="My Panel" iconCls="icon-save" collapsible="true">
	The panel content
</div>

当用标签创建组件时，kui-options属性可以用来支持HTML5兼容的属性名。 因此你可以从写上面的代码： 

<div id="p" class="kui-panel" style="width:500px;height:200px;padding:10px;"
	 title="My Panel" kui-options= "iconCls:'icon-save',collapsible:true>
	The panel content
</div>

下面的代码展示如何创建一个combobox组件并且绑定onSelect事件。 

<input class="kui-combobox" name="language"
		kui-options="
			url:'combobox_data.json',
			valueField:'id',
			textField:'text',
			panelHeight:'auto',
			onSelect:function(record){
				alert(record.text)
			}" />
